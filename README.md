# Projet Java Logs

## Objectif
Analyser des fichiers de logs Apache et afficher des rapports.

## Java
Pour lancer :
1. `javac *.java`
2. `java Main`
3. Optionnel : préciser un fichier de logs, par exemple : `java Main ressources/access_log_clean.txt`

## Web
Ouvrir `index.html` dans un navigateur et charger un fichier de logs au format Apache.
